# 🚀 `Quick Start`

### 💿 Installation

```bash
npm i @mohitkothari/buttons
```

or

```bash
yarn add @mohitkothari/buttons
```

### 🧰 Usage

```jsx
import { Button } from "@mohitkothari/buttons";

const App = () => (
    <>
        <Button label="Submit" size="small" backgroundColor="#00d2d3" textColor="#fff" />
    </>
);
```
